Talk

Zooniverse Talk is a forum replacement

Object Oriented Discussion

  • You can talk about a Focus
    • Assets, Groups, and Collections (AssetSets and KeywordSets) are Focii
    • They have one twitter-style Conversation and many in-depth Discussions. Both are really just Discussions
    • Focii and their Discussions have ZooniverseIds that are unique in the Zooniverse
    • Focii can have Tags, which are created by Users in Comments
  • Discussions are really just containers for Comments
    • Then again they also store lots of other metadata
    • They also get the Tags from their Comments
  • If a User doesn't want to talk about a Focus, they can go to the Boards
    • A Board is some category of conversation. We supply 'help', 'science', and 'chat'
    • Moderators and admins can create SubBoards to help Users organize discussion
  • An Asset is whatever the Zoo is classifying, it's really the primary object of discussion
  • A Collection is a User created set of Assets
    • An AssetSet is a manually curated set of Assets
    • A KeywordSet is a dynamic set of Assets based on the Tags specified by the User
  • A Group is a system managed group of Assets
    • Imagine a book with many pages. The Asset is the page, the Group is the Book
    • Or how about a CD with many songs. The Asset is the song, the Group is the CD
  • A User is... well... a User
    • They can send Messages to each other too

Getting started

  • Setup RVM, probably with some packages depending on your environment
  • Install Ruby 1.9.2
  • Install RubyGems
  • Install MongoDB
  • Make sure Bundler is installed
  • Run bundle from the app to install the gem dependencies

Configuring

  • Setup config files in Talk/config
    • Examples are show in Talk/config/examples
  • Configure mailer settings in Talk/environments/*
  • Configure your Capistrano deploy script at Talk/config/deploy.rb
